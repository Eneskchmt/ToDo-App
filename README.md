# ToDo App Project

<p>
  Todo App, günlük görev takibini kolaylaştırmak için tasarlanmış, modern ve karanlık tema (dark mode) odaklı bir mobil uygulama arayüzü projesidir. Minimal tasarımı, ilerleme göstergesi ve sade etkileşimleriyle kullanıcıya odaklanmış bir deneyim sunar.
</p>

<img src="./gif/todo.gif">

## Öne Çıkan Özellikler 

<ul>
  <li>Günlük görev listesi (Today’s Tasks)</li>
  <li>Tamamlanan görev sayısına göre ilerleme çubuğu (% progress)</li>
  <li>Yeni görev ekleme alanı (input + add butonu)</li>
  <li>Alt tab bar navigasyonu (Todos / Settings)</li>
  <li>Mobil ekranlara tam uyumlu tasarım</li>
  <li>Boş durum (empty state) tasarımı (“No todos yet!”)</li>
</ul>

## Kullanılan Kütüphaneler ve Teknolojiler 

<ul>
  <li>React Native</li>
  <li>Expo</li>
  <li>Convex DB</li>
  <li>JavaScript</li>
  <li>TypeScript</li>
  <li>Icon Library</li>

</ul>
